# 배포 링크

## 프로젝트 소개
원티드 프리온보딩 프론트엔드 코스에서 **정렬, 날짜 기능**을 구현한 과제입니다.
저희 팀은 정렬에 합병 정렬(merge sort)를 사용하였습니다. 이유는 다음과 같습니다.

- 병합 정렬의 시간 복잡도는 O(nlog₂n)으로 기본 정렬 알고리즘에 비해 성능이 좋은 편에 속합니다. 
- 또한 이는 어떠한 경우에도 동일하게 적용됩니다. (안정적)


## 설치 및 실행
`npm install -> npm start`
